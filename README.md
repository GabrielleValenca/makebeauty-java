# MakeBeauty - Sistema Java com MySQL

Projeto desenvolvido em Java para gerenciamento de produtos de beleza.

O sistema permite cadastrar, listar, atualizar e deletar produtos, além de simular um carrinho de compras.

## Tecnologias utilizadas

- Java
- MySQL
- JDBC
- Programação Orientada a Objetos

## Funcionalidades

- Cadastro de produtos
- Listagem de produtos
- Atualização de produtos
- Remoção de produtos
- Carrinho de compras
- Cálculo do valor total

## Estrutura do projeto

model → classes do sistema  
dao → acesso ao banco de dados  
database → conexão com MySQL  
Main → execução do sistema
