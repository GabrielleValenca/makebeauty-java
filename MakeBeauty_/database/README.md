# MakeBeauty

Projeto de banco de dados para organização de produtos de beleza.

## Tecnologias
- MySQL
- SQL

## Estrutura do banco

O banco possui uma tabela principal chamada `makeBeauty` que armazena informações gerais do produto.

As demais tabelas especializam os produtos em categorias:

- cuidado_capilar
- maquiagem
- cuidado_corporal

## Como executar

1. Abra o MySQL
2. Execute o script localizado em:

database/makebeauty.sql
